Portfolio

This is a selection of works from my time at university. Please, feel free to click the images as they are links which will take you to a video or web page of the project.

PoweRoutage

PowerRoutage is a video game 'vertical slice' prototype I developed in the course Video Game Design along with students Philip Berrez, Kári Steinn Aðalsteinsson and Sindri Pétursson during April-June 2022. I was mainly responsible for the programming, and developed the player character's functions, the 'cable' physics and procedural graphics, and more. The game was made in the Godot game engine and coding was done in its proprietary scripting language GDScript.

Explosion computer graphics effect

This is a computer graphics explosion effect which I developed in the course Interactive Computer Graphics course during the summer 2022. It was made in the Godot game engine using the OpenGL shading language.

Watchbook

Watchbook is a webpage I developed with students David Touma and Edris Hakimzada in October-December 2020. It allows users to log in and create 'watch lists' of movies by querying the movie.db database. I was responsible for the back-end and database API implementation.

Plant vs Birds

Plant vs Bird is an AR (alternate reality) mobile game which I developed with students Sandra Larsson, Jessie Liu, and Oliver Kamruzzaman from October 2022 to January 2023. I was the main programmer; the game was made in the Unity game engine and programmed in C#.
